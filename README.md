# pet-project-KPI-analysis
SQL-анализ данных реальной команды поддержки клиентов. Датасет охватывает 86 cотрудников и более 22 000 завершённых тикетов travel tech компании.

Как использовать:
--  1. нужно открыть сайт sqliteonline.com
--  2. вставить весь файл и нажать Run (таблицы создадутся сразу)
--  3. затем выполнить запросы З1–З5 по одному

Первая часть файла содержит создание таблиц и заполнение данных. Во второй части находятся SQL-запросы для анализа.
